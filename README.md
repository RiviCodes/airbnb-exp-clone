# Airbnb Online Experiences Clone

### [Live on Netlify 🚀](https://rivicodes-airbnb-exp-clone.netlify.app/)

_Single Page Application aimed to mimic the appearance of Airbnb's Online Experiences site._

### Pre-requirements 📋
